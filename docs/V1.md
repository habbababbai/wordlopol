# Wordlopol 1.0

First playable release — Polish Wordle-style daily and infinite modes, email auth, profile stats.

**Design reference:** [Figma Make — Wordlopol](https://www.figma.com/make/RggardTz2oWCfly04e4HdX/Wordlopol)

Package versions at release: `@wordlopol/api` **1.0.0**, `@wordlopol/web` **1.0.0** (HTTP API stays `/v1`).

## Shipped

### Repo & tooling

- pnpm 11 workspaces + Turborepo, ESLint, Prettier, Husky, GitHub Actions CI
- Supply-chain hardening in `pnpm-workspace.yaml` — see [SUPPLY_CHAIN.md](./SUPPLY_CHAIN.md)
- Shared package: `evaluateGuess`, date pickers, API DTOs (`@wordlopol/shared`)

### API (`apps/api`)

- Auth: register, verify email, login, refresh, logout, logout-all, password reset/change, email change, account delete
- Daily: `GET /v1/daily/today`, `POST /v1/daily/guess` (guest + registered; guest anti-cheat)
- Infinite: `GET /v1/infinite/next`, `POST /v1/infinite/guess` (verified users)
- Profile: `GET /v1/user/profile` with stats
- Routes under `/v1`; stable error codes; OpenAPI spec + Postman collections
- Health (`/health`, `/v1/health`), Zod validation, integration + e2e tests

### Web (`apps/web`)

- Design tokens, light/dark theme, UI primitives, `/dev/ui`
- App shell, routing, Polish UI (`locales/pl.json`)
- Home page (Figma layout, auth-aware CTAs and nav)
- API client (401 refresh, CSRF, `@wordlopol/shared` types)
- TanStack Query for server I/O; session bootstrap; `useAuth()`
- Auth: login, register, verify-email, forgot/reset password
- `ProtectedRoute` + verified-email guard; profile with stats
- Settings: display name, change password/email, logout-all, delete account
- Game: `GameBoard`, `PolishKeyboard`, daily (guest + registered), infinite (verified)
- Result modal after win/loss; optional typing/reveal sounds; `GamePageHeader`, `GameStatusBar`
- A11y: responsive board, keyboard/nav focus, skip link, `prefers-reduced-motion`
- Vitest + Testing Library in `src/__tests__/`

### Data

- `data/words.txt` — 5-letter Polish dictionary (import via `pnpm db:import-words`)

## Pre-release QA (manual)

Check before tagging **1.0.0**:

- [ ] Guest daily play and registered daily stats persist correctly
- [ ] Infinite mode gated to verified users
- [ ] Settings → logout-all revokes sessions
- [ ] 320px viewport — board fits without horizontal scroll
- [ ] Tab focus visible on keyboard and nav; skip link works
- [ ] `prefers-reduced-motion` disables tile/ambient animations and sounds

## Out of scope for 1.0

| Feature                       | Notes                                                |
| ----------------------------- | ---------------------------------------------------- |
| Timed mode                    | 5-minute sprint; schema has `TIMED` enum, no API yet |
| Multiplayer                   | WebSocket lobbies, up to 4 players                   |
| Mobile app                    | React Native + Expo (`apps/mobile`)                  |
| English UI                    | Polish only for v1                                   |
| Leaderboards                  | Stats on own profile only                            |
| Answer/guess dictionary split | Single `Word` table for v1                           |
