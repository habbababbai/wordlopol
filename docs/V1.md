# v1.0 scope

What ships in the first playable release vs what is still missing.

**Design reference:** [Figma Make — Wordlopol](https://www.figma.com/make/RggardTz2oWCfly04e4HdX/Wordlopol)

## Done

### Repo & tooling

- pnpm workspaces + Turborepo, ESLint, Prettier, Husky, GitHub Actions CI
- Shared package: game logic + API DTOs used by api and web

### API (`apps/api`)

- Auth: register, verify email, login, refresh, logout, logout-all, password reset/change, email change, account delete
- Daily: `GET /today`, `POST /guess` (guest + registered)
- Infinite: `GET /next`, `POST /guess` (verified users)
- Profile: `GET /user/profile` with stats
- Health check, Zod validation, integration + e2e tests

### Web (`apps/web`)

- Design tokens, light/dark theme, UI primitives, `/dev/ui`
- App shell, routing, Polish i18n (`locales/pl.json`)
- Home page (Figma layout, auth-aware CTAs and nav)
- API client (401 refresh, `@wordlopol/shared` types)
- TanStack Query for all server I/O; Zustand `sessionChecked` bootstrap; `useAuth()` reads session cache
- Auth forms: login, register, verify-email, forgot/reset password
- `ProtectedRoute` + verified-email guard; profile page with stats
- Vitest tests in `src/__tests__/`
- Game UI: `GameBoard`, `PolishKeyboard`, daily page (guest + registered play)
- Game **route stubs** (infinite/settings — titles only, no play UI yet)

## Remaining for v1.0

### Auth UI

- [ ] Settings: change password/email, logout-all, delete account

### Game UI

- [ ] Infinite page — fetch next word, submit guesses

### Ship quality

- [ ] Responsive layout, focus/ARIA, `prefers-reduced-motion`
- [ ] Manual pass: guest daily, registered stats, infinite gated correctly

## Out of scope for v1.0

Deferred until after first release:

| Feature      | Notes                                                |
| ------------ | ---------------------------------------------------- |
| Timed mode   | 5-minute sprint; schema has `TIMED` enum, no API yet |
| Multiplayer  | WebSocket lobbies, up to 4 players                   |
| Mobile app   | React Native + Expo (`apps/mobile`)                  |
| English UI   | Polish only for v1                                   |
| Leaderboards | Stats on own profile only                            |
